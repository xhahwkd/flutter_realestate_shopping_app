## 프로젝트 개요
### "임박세일 홈즈"는 부동산 급매 매물을 실시간으로 연결하는 매물 중심 쇼핑 플랫폼입니다.
#### 이런 분들을 위해 만들었습니다.

> 매물을 빠르게 팔고 싶은 집주인
> 

> 좋은 조건의 급매를 찾는 실수요자
> 

> 투자 타이밍을 노리는 현명한 투자자
>
